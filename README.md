Eduup is an online learning platform built with Next.js.

## Website

[https://eduup.inqtube.com/](https://eduup.inqtube.com/)

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open the app locally in your browser.

## Branding

This project is branded as `Eduup`. App-facing URLs and legacy brand references have been replaced with `https://eduup.inqtube.com/`.
